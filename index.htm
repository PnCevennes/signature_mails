Générer sa signature

Il suffit de remplir le formulaire avec les informations demandées.

Le nom, le prénom et la fonction peuvent être écrits en minuscule. Ils seront reformatés pour s'afficher joliment.

La ligne "Téléphone" se remplit sous la forme "Tel : +33 (0)1 23 45 67 89"
Il est important de bien mettre l'indicatif Fr (+33) et de séparer les chiffres par paire séparées par une espace.

Afin de bien respecter la norme d'affichage, il est demandé de séparer les différents numéros de téléphone par un " - "

Appuyer ensuite sur le bouton vert "Générer la signature Thunderbird" pour enregistrer le fichier signature (sélectionner "enregistrer le fichier")

Le fichier sera alors enregistré dans votre dossier "Téléchargements" sous le nom "signature_prenom_nom.htm".
Il est recommandé de déplacer ce fichier dans un dossier différent.

Comment ajouter sa signature à Thunderbird

  1. Cliquer sur votre adresse e-mail (au dessus du dossier "Courrier entrant").
  2. Sur la page affichée, cliquer sur "Voir les paramètres pour ce compte".
  3. Dans la fenetre qui s'ouvre, cocher la case "Apposer la signature à partir d'un fichier, puis sélectionner le fichier que vous venez d'enregistrer.

Comment ajouter sa signature à Zimbra

La signature zimbra est une version allégée de la signature thunderbird. Sans les différents logos des réseaux sociaux.

  1. Sur cette page, cliquez sur le bouton "Générer la signature Zimbra" puis choisissez de l'ouvrir avec votre navigateur
  2. Dans Zimbra, cliquez sur l'onglet "Préférences", puis dans le menu de gauche, sur "Signatures"
  3. Dans la fenetre qui apparait, sur la droite, cliquez sur le bouton "format texte simple" et sélectionnez "format HTML"
  4. Ouvrez le fichier téléchargé (ou simplement sur cette page, rappuyez sur "Générer la signature" et sélectionnez l'option ouvrir avec votre navigateur).
  5. La page "signature" affichée, séléctionnez tout (ctrl-A), copiez (ctrl-C) et collez dans la zone de texte de zimbra (ctrl-V), en supprimant l'existant si nécessaire. Réglez l'utilisation des signatures, puis, pour valider les modifications, appuyer sur "Enregistrer".
