Générer sa signature

Il suffit de remplir le formulaire avec les informations demandées.

Le nom, le prénom et la fonction peuvent être écrits en minuscule. Ils seront reformatés pour s'afficher joliment.

La ligne "Téléphone" se remplit sous la forme "Tel : +33 (0)1 23 45 67 89"
Il est important de bien mettre l'indicatif Fr (+33) et de séparer les chiffres par paire séparées par une espace.

Afin de bien respecter la norme d'affichage, il est demandé de séparer les différents numéros de téléphone par un " - "

Appuyer ensuite sur le bouton "Enreg." pour enregistrer le fichier signature (sélectionner "enregistrer le fichier")

Comment ajouter sa signature à Thunderbird

  1. Cliquer sur votre adresse e-mail (au dessus du dossier "Courrier entrant").
  2. Sur la page affichée, cliquer sur "Voir les paramètres pour ce compte".
  3. Dans la fenetre qui s'ouvre, cocher la case "Apposer la signature à partir d'un fichier, puis sélectionner le fichier que vous venez d'enregistrer qui doit se trouver dans votre dossier "Téléchargements" sous le nom "signature_prenom_nom.htm".

Il est recommandé de copier ou déplacer le fichier signature hors du dossier "Téléchargements".

Comment ajouter sa signature à Zimbra

  1. Cliquer sur l'onglet "Préférences", puis dans le menu de gauche, sur "Signatures"
  2. Dans la fenetre qui apparait, sur la droite, cliquer sur le bouton "format texte simple" et sélectionner "format HTML"
  3. Ouvrez le fichier téléchargé (ou simplement sur cette page, rappuyez sur "Enreg." et sélectionnez l'option ouvrir avec votre navigateur).
  4. La page "signature" affichée, tout séléctionner (ctrl-A), copier (ctrl-C) et coller dans la zone de texte de zimbra (ctrl-V), en supprimant l'existant si nécessaire. Régler l'utilisation des signatures, puis, pour valider les modifications, appuyer sur "Enregistrer".
